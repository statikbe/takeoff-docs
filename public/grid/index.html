Introductie

Het framework bevat een responsive, mobile first fluid grid systeem met 12 kolommen.

Het grid systeem wordt gebruikt om een layout te creëren met rijen en kolommen. Enkele weetjes voor het gebruik ervan:

  • Rijen moeten binnen een .container geplaatst worden.
  • Elke rij moet de .grid class meekrijgen. Een .grid__* element kan niet bestaan zonder een omvattend .grid element.
  • Content moet in een kolom (.grid__*) geplaatst worden, een kolom is het enige toegelaten child-element van een rij.
  • Gebruik rijen om een horizontale groepen van kolommen te creëren.
  • De ruimte tussen de kolommen wordt bepaald met een padding. Om die reden heeft elke rij een negatieve marge links en rechts met de breedte van die padding.
  • Omdat onze grid mobile first werkt, werken alle classes zonder een breakpoint (*bp-med*) definitie vanaf een minimum breedte van 0px. Wanneer er in de class een breakpoint wordt gedefineerd, werkt die class vanaf het breakpoint en hoger. (Meer info over de breakpoints vind je in deze tabel.
  • Om kolommen even hoog te maken (bijvoorbeeld bij .media--cover) kan je .grid--flex toevoegen aan de rij class. Let op! Dit maakt enkel je kolommen even hoog, niet de inhoud binnen de kolommen.

Het grid bevat twee soorten grid element:

Grid met gelijke kolommen

  • Het aantel kolommen wordt gedefinieerd bij de rij (.grid), .grid grid--bp-sml-2-col grid--bp-lrg-3-col heeft bijvoorbeeld van het sml-breakpoint 2 kolommen en vanag het lrg-breakpoint 2 kolommen.
  • Elke kolom heeft enkel de .grid__item class

Grid met ongelijke kolommen

  • De breedte van de kolom wordt gedefinieerd op de kolom zelf (.grid__1~12). Het grid bevat 12 kolommen, in de class kan je kies je hoeveel kolommen deze kolom breed moet zijn. Ook hier zijn er responsive classes beschikbaar: grid__12 grid--bp-med__6 grid--bp-lrg__4. In dit voorbeeld heb je standaard een kolom die 12 kolommen bevat, wat overeenkomt met 1 kolom binnen de rij. Bij het med-breakpoint overspant deze kolom maar 6 kolommen van de 12, en vanaf het lrg-breakpoint maar 4.
  • Je kan met dit systeem ook gelijke kolommen maken, maar het is aangewezen om het systeem met gelijke kolommen hiervoor te gebruiken.

xsm
≥ 440px
sml
≥ 660px
med
≥ 800px
